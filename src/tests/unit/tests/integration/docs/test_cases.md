# Login Test Cases

## TC-01
**Description:** Verify login with valid credentials  
**Pre-condition:** User account exists  
**Input:** Correct username and password  
**Steps:**
1. Open login page
2. Enter valid credentials
3. Click login  
**Expected Result:** User logged in successfully

## TC-02
**Description:** Login redirects to dashboard  
**Expected Result:** Dashboard opens

## TC-03
**Description:** Login with correct username  
**Expected Result:** Login success

## TC-04
**Description:** Remember me works  
**Expected Result:** User remains logged in

## TC-05
**Description:** Login with wrong password  
**Input:** Wrong password  
**Expected Result:** Error message shown

## TC-06
**Description:** Login with empty fields  
**Input:** Empty username & password  
**Expected Result:** Validation error
